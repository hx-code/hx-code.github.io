面试题

1.说一说let const var 的区别？

  • 变量提升
  • 暂时性死区
  • 块级别作用域
  • 重复声明
  • 修改声明的变量
  • 使用

变量提升

var声明的变量,即变量可以在声明之前调用，值为undifined

let和const不存在变量提升，即他们所声明的变量一定要在声明后使用

否则报错

 //var 
 console.log(a)  //undifined
 var a  = 10

 //let
 console.log(b) //变量不会提升g
 let b = 10

 //const
 console.log(b) //变量不会提升
 const c = 10

暂时性死区

var不存在暂时性死区

let和const存在暂时性死区,只有等到变量声明的那一行代码出现

才可以获得和使用该变量

//var
console.log(a) //undefined
var a = 10

//let
console.log(b) //Cannot acess 'b' before initialization
let b = 10

//const
const.log(c)
let c = 10

块级作用域

var不存在块级作用域

let和const存在块级作用域

//var 
{
    var a = 20
}
console.log(a)
//let
{
    let b = 20 // b is not defined
}
console.log(b)
//const
{
    const c = 20
}
console.log(c)

修改声明变量

var和let可以

const声明只读一个变量，一旦声明常量不能改变

//var
var a = 10
a = 20
console.log(a)
//let
let b = 10
b = 20
console.log(b)
// const
const c =19
c= 20
console.log(c)

重复声明

var允许重复声明
let和const在同一作用域不许重复变量声明

//var
var a = 10
var a = 20 //20

//let
let b =10
let b =20

//const
const c  = 10
const c = 20 //变量已经声明
